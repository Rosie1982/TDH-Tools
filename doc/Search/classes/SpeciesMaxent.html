Extend this class to create jobs / actions to run in the GRID - Allows you to create a action to run without having to worry about the infrastucture of status check and passing data into and out of the HPC

Command Actions can be initialise, queued, executed and returned to the swebserver

package CommandAction\Species\SpeciesMaxent Wrapper for Maxent.jar and assocoiated projections
